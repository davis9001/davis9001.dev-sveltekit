After successully helping to add a dark/light theme toggle feature to the Deno
Docs documentation website I ended up using the Deno Fresh documentation site
(which still did not have dark mode) on a regular basis. I decided to contribute
back to the Deno Fresh project by adding dark mode support as I did with Deno
Docs.

In this case I was a bit more experienced with the process and the Deno team PR
response. My pull request was merged in without first being duplicated by a
staff member.

You can see the full history in these two pull requests:

- Merged Pull Request: https://github.com/denoland/fresh/pull/2820
- Followup Pull Request to fix mobile and other minor issues:
  https://github.com/denoland/fresh/pull/2821
