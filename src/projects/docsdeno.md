After the Deno 2.0 launch I started using Deno a lot and noticed the docs site
did not have a dark theme option. I found a matching issue on GitHub that was
gaining some support so I decided to get the ball rolling on it.

After about a couple of long nights getting things close enough to show off I
submitted a pull request which was used in the final merge to the public site.
Woohoo!
